# Informe Técnico - Avance 2

## Resumen Ejecutivo
Sistema de gestión digital para pedidos educativos implementado para la Municipalidad de Fernández.

## Arquitectura del Sistema
- **Frontend**: AppSheet para registro móvil
- **Base de Datos**: Google Sheets como almacenamiento central
- **Analytics**: Looker Studio para dashboards
- **Demo**: Streamlit para demostración interactiva

## Métricas de Calidad
- Reducción tiempo registro: 88% (25min → 3min)
- Mejora calidad datos: 94% (35% → 2% errores)
- Cobertura análisis: 100% disponible vs 0% antes

## Tecnologías Utilizadas
- Google Workspace (Sheets, AppSheet, Looker Studio)
- Python (Pandas, Streamlit, Matplotlib)
- GitHub para control de versiones
