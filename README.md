# Gestion-Educativa-Fernandez
Sistema de Gestión de pedidos educativos - Equipo Municipalidad Fernández

[![Haz clic para ver el video en YouTube](https://img.youtube.com/vi/_M9ujT7srLw/0.jpg)](https://www.youtube.com/watch?v=_M9ujT7srLw)

========== Contexto General ======

La aplicación EduPedidos fue diseñada para optimizar la carga, seguimiento y gestión de solicitudes provenientes de instituciones educativas de la ciudad de Fernández.
El objetivo principal es mejorar la organización interna y agilizar el trabajo del personal de la Dirección de Educación mediante un sistema centralizado, accesible y
seguro.

=========== Descripción ================

EduPedidos fue desarrollada utilizando AppSheet, integrándose directamente con Google Sheets como base de datos principal.
El sistema permite:
* Registrar solicitudes educativas
* Gestionar y editar pedidos
* Controlar permisos de acceso
* Visualizar datos de forma organizada y actualizada

======== Usos y Permisos ===========

Para acceder correctamente a la aplicación:
* El usuario debe contar con una cuenta de correo electrónico válida (Gmail recomendado).
* Esa cuenta debe estar registrada en el panel de usuarios autorizados de AppSheet.
* Solo los usuarios habilitados podrán ingresar con permisos de: Lector / Editor.

*AVISO IMPORTANTE!!!*: Antes de utilizar la app, comuníquese con el equipo de desarrollo para registrar su correo y habilitar su acceso.


======= Requisitos =========

Para la Versión WEB

**Requisitos Mínimos:**
Procesador: Dual core o Superior.
Memoria RAM: 2 GB o Superior.
Video (Integrado o Dedicado): 512 Mb o Superior.
Conexión a Internet de 5 mbps o más (Conexiones inferiores pueden generar problemas al cargar datos, o que directamente no se muestre 
la interfaz gráfica de la aplicación).
Sistema Operativo: Linux (En todas sus distribuciones), Windows 7 o Superior.
Navegadores compatibles: Brave, Chrome, Edge, Firefox, Opera, Safari.

*Conexiones lentas pueden generar demoras o impedir la carga correcta de la interfaz.*

Para la Versión APP MOVIL

**Requisitos Mínimos:**

Celular o dispositivo movil con Android versión 4.5 o superior.

Conexión a internet estable.

*AVISO IMPORTANTE!!!*: Ante cualquier inconveniente en la aplicación comuniquese inmediatamente con el equipo de desarrollo.


========= Manual de Usuario ===============

En la presente guía de usuario se detallará a continuación los procedimientos o pasos correctos para el uso de la aplicación:

Para la versión WEB: 
* Antes de iniciar la aplicación, asegurese de haber ingresado en su cuenta de correo electronico, o tener a mano su 
usuario y contraseña. 
* Dirijase a la pagina de la aplicación mediante la siguiente URL: https://www.appsheet.com/start/cee86596-5ac0-4969-a496-8412fdbe4edf
* Ingrese con su correo autorizado.
* Siga las instrucciones en pantalla para acceder al panel principal.


Para la versión APP MOVIL: 
* Si es la primera vez que ingresa a la aplicación, primero asegurese de ir a google playstore y descargar la 
aplicación llamada APP SHEETS.
* Una vez descargada en su dispositivo movil, asegurese de que se instaló correctamente.
* Abra la aplicación buscando el icono de AppSheet en su pantalla del dispositivo.
* Acceda con el correo registrado por el equipo de desarrollo.
* Seleccione la app EduPedidos dentro del panel de aplicaciones disponibles.
