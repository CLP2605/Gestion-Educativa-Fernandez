# Gestion-Educativa-Fernandez
Sistema de Gestión de pedidos educativos - Equipo Municipalidad Fernández

======================
     Contexto General
======================

Esta aplicación fue diseñada por el Grupo 3 - de la Sede Fernández, con el objetivo de cargar y gestionar todas las solicitudes de 
instituciones educativas de una manera más eficiente, mejorando así ampliamente la capacidad de trabajo del personal de la Dirección de 
Educación de la ciudad de Fernández.

=======================
      Descripción
=======================

La aplicación EduPedidos fue diseñada mediante uso de APPSHEETS, está conectada con Google Sheets y funciona como un sistema de 
Gestión y manejo de datos (Solicitudes). 

=======================
    Usos y Permisos
=======================
Para el correcto funcionamiento de la aplicación es necesario contar con una cuenta de correo electronico, y que esta sea correctamente
registrada dentro de la base de datos de la aplicación APPSHEETS para que se logre una correcta conectividad del usuario final.
La aplicación cuenta con permisos de Lector y Editor solamente para los usuarios debidamente registrados con anterioridad.

*AVISO IMPORTANTE!!!*: Notificar al equipo  de desarrollo para un correcto registro dentro de la aplicación.


=======================
    Requisitos
=======================

Para la Versión WEB

**Requisitos Mínimos:**
Procesador: Dual core o Superior.
Memoria RAM: 2 GB o Superior.
Video (Integrado o Dedicado): 512 Mb o Superior.
Conexión a Internet de 5 megas o más (Conexiones inferiores pueden generar problemas al cargar datos, o que directamente no se muestre 
la interfaz gráfica de la aplicación).
Sistema Operativo: Linux (En todas sus distribuciones), Windows 7 o Superior.
Navegadores compatibles: Brave, Chrome, Edge, Firefox, Opera, Safari.

Para la Versión APP MOVIL
**Requisitos Mínimos:**
Celular o dispositivo movil con Android versión 4.5 o superior.

*AVISO IMPORTANTE!!!*: Ante cualquier inconveniente en la aplicación comuniquese inmediatamente con el equipo de desarrollo.


=========================
  Manual de Usuario
=========================

En la presente guía de usuario se detallará a continuación los procedimientos o pasos correctos para el uso de la aplicación:

Para la versión WEB: Antes de iniciar la aplicación, asegurese de haber ingresado en su cuenta de correo electronico, o tener a mano su 
usuario y contraseña. Dirijase a la pagina de la aplicación mediante la siguiente URL: https:\\ejemplolaaplicaciónaqui.com. 

Una vez dentro de la pagina direccionada ingrese apretando en ...

Para la versión APP MOVIL: Si es la primera vez que ingresa a la aplicación, primero asegurese de ir a google playstore y descargar la 
aplicación llamada APP SHEETS, una vez descargada en su dispositivo movil, asegurese de que se instaló correctamente.
Abra la aplicación buscando el icono de AppSheet en su pantalla del dispositivo.
